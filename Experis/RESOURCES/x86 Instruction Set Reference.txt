
  x86 Instruction Set Reference

Derived from the September 2014 version of the /Intel® 64 and IA-32
Architectures Software Developer’s Manual/, volumes 2A and 2B.

More info at zneak/x86doc <https://github.com/zneak/x86doc>

*This reference is not perfect.* It's been mechanically separated into
distinct files by a dumb script. It may be enough to replace the
official documentation on your weekend reverse engineering project, but
in doubt, go get the official and freely available documentation.

Download documentation set <http://www.felixcloutier.com/x86/x86.zip>
